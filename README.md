# Football_Scores_Predictor
## Overview
This is a statistical forecasting model for predicting game outcomes in the German Bundesliga. The application compares different Machine Learning classifiers such as Naïve Bayes and Decision Trees. 
The highest accuracy is Gaussian Naïve Bayes, ~75% 
The dataset for this application can be found in the project directory. 
## Dependencies
* Scikit-learn
* Pandas

## Usage
Run `jupyter notebook` in terminal.
